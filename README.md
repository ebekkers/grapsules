# grapsules
E(d) equivariant geometric capsules

```
conda create --yes --name grapsules python numpy scipy matplotlib
conda activate grapsules
conda install pytorch=1.9 torchvision torchaudio cudatoolkit=11.1 -c pytorch -c conda-forge -y
conda install gcc_linux-64 -y
conda install pytorch-geometric -c rusty1s -c conda-forge -y
pip3 install wandb
pip3 install pytorch_lightning
pip3 install pandas
pip3 install timm

```
